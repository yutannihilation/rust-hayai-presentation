FFI(Foreign Function Interface)

  • あるプログラミング言語から他のプログラミング言語で定義された関数などを利用するための機構
  • つまりRcppみたいな？？（あんまり理解してない）

各言語のFFI

  • Ruby: Ruby-FFI
  • Python: CFFI
  • Node.js: node-ffi
  • R: Rffi

いよいよRからRustの関数をつかう

こんな感じのRustの関数を書く

#[no_mangle]
pub extern fn fib(n: u32) -> u32 {
    if n == 0 || n == 1 {
        n
    } else {
        fib(n - 1) + fib(n - 2)
    }
}

いよいよRからRustの関数をつかう

こんな感じのCargo.tomlを置く

[package]
name = "fib"
version = "0.1.0"
authors = ["yutannihilation <XXXXXXXX@example.com>"]

[lib]
name = "fib"
crate-type = ["dylib"]

いよいよRからRustの関数をつかう

こんな感じのRの関数をつくる

fib <- function(x){
  Rffi::callCIF(
    cif = Rffi::prepCIF(retType  = Rffi::uint32Type, argTypes = list(Rffi::uint32Type)),
    sym = "fib",
    x
  )
}
